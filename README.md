# BLT Manager Final

Bản cuối đã tách file, hạn chế lỗi vá chồng chéo.

## Cách dùng
1. Giải nén file zip.
2. Upload toàn bộ thư mục lên GitHub.
3. Giữ/copy lại dữ liệu cũ: `hoc_vien.csv`, `lich_ban.csv`, `hoc_phi_thay.csv` vào thư mục gốc hoặc `data/`.
4. Streamlit Cloud chọn file chính: `app.py`.

## Tính năng
- Dashboard báo cáo tổng quát.
- Lịch dạy realtime giờ Việt Nam.
- Điểm danh 12 buổi tự ghi ngày.
- Kết thúc khóa tự ẩn khỏi lịch.
- Học phí và tiền giáo viên.
- Cài đặt lịch bận.
