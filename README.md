# BLT Swimming Club Manager v2

## Chạy local
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy Streamlit Cloud
Upload toàn bộ project lên GitHub rồi chọn file `app.py`.

Dữ liệu được lưu ở root:
- `hoc_vien.csv`
- `lich_ban.csv`

Ảnh:
- `assets/logo.png`
- `assets/banner.png`
